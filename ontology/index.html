Preferred Namespace Uri

Description

An extension of RiC-O (Records in Contexts Ontology) designed to represent the complexity of born-digital archival materials, addressing requirements for technical metadata, digital provenance, integrity verification, and contextual relationships.

Technical metadata c

IRI http://w3id.org/bodi#TechnicalMetadata
Description

The actual value or content of technical metadata associated with a digital resource. For example, in 'cp:revision = 1', the value '1' represents the TechnicalMetadata instance, while 'cp:revision' is the Technical Metadata Type. These values describe specific technical characteristics such as format parameters, creation settings, and other measurable attributes.
Sub Class Of rico:Concept c

Technical Metadata Type c

IRI http://w3id.org/bodi#TechnicalMetadataType
Description

The type, property, or category that defines what kind of technical metadata is being described. For example, in 'cp:revision = 1', 'cp:revision' represents the TechnicalMetadataType that categorizes the metadata as revision information. This classification enables systematic organization of different kind of metadata like version identifiers, format specifications, encoding parameters, and preservation characteristics.
